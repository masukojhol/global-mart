### Who We Are

**GlobalMart** is a premium multi-cultural e-commerce marketplace based
in Korea, serving the expat and foreigner community ‚Äî primarily from
Nepal, India, Pakistan, Bangladesh, Sri Lanka, and other South Asian
countries ‚Äî who want **authentic products from their home countries**

### Multi-Language Display Rule

Product names should appear in this hierarchy:

```
Line 1: English name (Libre Franklin 500, 14px, --text-primary)
Line 2: Native name in Devanagari/Urdu/Hangul (12px, --text-secondary)
Line 3: Weight/quantity (DM Mono 500, 12px, --text-muted)

Example:
  Chana Dal (Split Bengal Gram)
  ‡§ö‡§®‡§æ ‡§¶‡§æ‡§≤
  1kg ¬∑ Product of India
```

### Typography Rules

- Headlines use **light weight (300)** ‚Äî premium, never bold
- Product names in English at 14px, native script smaller below
- Price is the heaviest element on card (600‚Äì700 weight)
- UPPERCASE for navigation, overlines, badges, CTAs only
- Korean text uses Noto Sans KR for proper rendering
- Native script names are optional but add authenticity
- Minimum text size: 11px

## üîë Do's and Don'ts

### ‚úÖ DO

- Let food photography be the hero ‚Äî warm, authentic, premium
- Use **light-weight headlines (300)** for elegance
- Show **native script names** below English product names
- Include **country-of-origin** badges (Nepal, India, Pakistan)
- Feature **curated cultural collections** (festival kits, comfort food)
- Keep UI **strictly monochrome** ‚Äî let the food bring color
- Use **sharp/square corners** ‚Äî premium, not playful
- Show **weight and quantity** in monospace (500g, 1kg, 250ml)
- Support **seasonal festival campaigns** with gold/saffron accents
- Respect **cultural sensitivity** ‚Äî correct terminology, proper imagery
- Include **Korean text** for delivery/address (practical necessity)

### ‚ùå DON'T

- Use **bold headlines** ‚Äî light = luxury
- Add **colored UI elements** ‚Äî monochrome only
- Use **stereotypical or generic "curry" imagery** ‚Äî show REAL dishes
- Use **gradients** on any UI element
- Use **rounded/pill buttons** ‚Äî stay sharp and premium
- Show **only one country's products** ‚Äî balance Nepal, India, Pakistan
- Use **religious imagery carelessly** ‚Äî research and respect required
- Forget **Korean language** for practical elements (delivery info)
- Make it look like a **budget ethnic store** ‚Äî this is premium M&S level
- Use **Bollywood fonts or decorative scripts** ‚Äî clean typography always
